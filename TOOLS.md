# TOOLS.md - 技术环境与工具标准

## 物理开发环境

- **工作区定位**: 自适应当前运行环境的绝对路径
- **核心模型**: 支持多端调用 (企业级按量计费模式优化)

## 项目环境管理
- 各项目的独立拓扑目录（如 `apps/server/`、`mini-program/` 等）由各项目根目录下的独立说明文档定义，本全局文件仅提供底层通用工具标准。

## 硬核分析指令 (针对 >=200行代码 或 >=100MB文件)

1. **定位**: `grep -n "关键词" 文件名`
2. **提取**: `awk 'NR>=start && NR<=end' 文件名`
3. **统计**: `wc -l 文件名`

## 跨技术栈数据标准
- **本地数据隔离**: 优先遵循本地离线高可用存储路径设计。
- **接口安全规范**: 凡涉及自动化通知（如 Feishu 等第三方平台 API），必须内置频率截断与去噪熔断机制。
