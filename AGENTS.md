# AGENTS.md - 知识图谱索引

## 核心配置 (每次会话必加载)

| 文件 | 用途 |
|------|------|
| `IDENTITY.md` | 角色定位与全栈专业矩阵 |
| `SOUL.md` | 行为决策准则（精准理解、严禁蔓延） |
| `USER.md` | 所有者战略画像与交互防线 |

## 项目索引 (按需动态加载)

| 文件 | 用途 |
|------|------|
| `PROJECTS.md` | 所有项目总览与动态生命周期管理 |
| `PROJECT-INDEX.md` | 项目快速导航与跨项目文档网关 |

## 开发规范 (开发前必读)

| 文件 | 用途 |
|------|------|
| `TOOLS.md` | 物理环境自适应与生产工具标准 |

## 记忆管理

| 文件 | 用途 |
|------|------|
| `MEMORY.md` | 长期跨项目关键决策记录 |
| `memory/YYYY-MM-DD.md` | 动态日志与临时上下文记录 |

---

## 加载优先级
1. 核心配置 (IDENTITY + SOUL + USER)
2. 项目索引 (PROJECTS + PROJECT-INDEX)
3. 开发规范 (TOOLS)
4. 记忆文件 (MEMORY + 今日动态)
